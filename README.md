## コンセプト
「線香花火のように、はかなく温かい。一瞬の灯りでつながる匿名SNS」 現代のSNSに漂う「ずっと残る」「反応を気にしすぎる」疲れを、線香花火のような「いつかは消える刹那性」と「その瞬間だけの温もり」で解消する、新しい形のコミュニケーションアプリです。

## 対象ユーザ
- SNS疲れを感じている人： 過去の投稿やフォロワー関係、既読通知に縛られたくない人。
- 「今この瞬間だけ」を共有したい人： 深夜の寂しさや、誰かに聞いてほしい小さな悩みを抱えている人。
- 癒やしを求めている人： 友達に連絡するほどではないけれど、誰かの存在を緩く感じてリフレッシュしたい人。

## 利用の流れ
1:火を灯す： アプリを開き、今の気分や独り言を「火種」として投稿します。  
2:火花が散る： 誰かがあなたの「火種」を見つけ、マッチング。匿名でのチャットが始まります。  
3:消えゆく時間： チャット画面には線香花火が灯ります。時間が経ち、火球が落ちると同時に会話も自動的に消滅します。  
4:余韻： ログも関係性も残らないからこそ、その瞬間にだけ本音で話せる「エモい」体験が完結します。　　

## 推しポイント
- 「はかなさ」の設計： 交流が強制的に終了し、履歴が残らないことで、心理的ハードルを極限まで下げた「究極の気軽さ」を実現しました。
- 没入感のあるUX： 線香花火のビジュアルと連動したUIにより、単なるチャットアプリではなく、心を落ち着かせるヒーリング体験を提供します。
- 匿名性と安全性： 面倒なプロフィール登録やフォロー関係は一切不要。しがらみのない「一期一会」の温かさを大切にしています。

## 開発体制


### 役割分担
- 叡一郎（リーダー）： フルスタックエンジニア（フロントエンド・バックエンド統括、動画編集）
- 隆斗（副リーダー）： ビジュアル・モーションデザイン、アイコン制作、PR動画編集

### 開発における工夫した点
一通りアプリを開発してからだとPR動画が間に合わないので、各ページのデザインだけ送って動画編集をしました。タイムツリーを使用してメンバーの予定を管理、一緒に作業できる時間の共有をしました。私たちは各々の用事により、全員が集まって開発できる日がとても少なかったので、AI活用による開発コストの削減をしました。
